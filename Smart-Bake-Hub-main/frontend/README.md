# Smart Bake Hub - Frontend

This is the frontend application for Smart Bake Hub, built using React, Vite, and Tailwind CSS.

## How to Run

1. Ensure the backend server is running (see the main project `README.md` in the root directory for full instructions).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to the URL provided in the terminal (usually `http://localhost:5173`).

For full project setup, database initialization, and backend configuration, please refer to the `README.md` file in the root of the project repository.
